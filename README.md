# hk-rooftop-greenness-scripts
Google Earth Engine and ArcGIS scripts for rooftop greenness extraction

This repository contains Google Earth Engine (GEE) scripts and ArcGIS Pro workflows used in:
"Satellite-based assessment of rooftop greenness dynamics and inequality in Hong Kong", 2025.

## Contents
- gee_scripts/: NDVI compositing, cloud masking, NDVI extraction
- arcgis_workflows/: Zonal statistics, data preprocessing, Rooftop greenness extraction, Spatial join

## How to use
GEE scripts can be copied directly into the Code Editor. 
ArcGIS arcpy script can be executed from the ArcGIS Pro Python window or as script tools in a toolbox.
